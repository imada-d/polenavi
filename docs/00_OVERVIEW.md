# PoleNavi（ポールナビ） - プロジェクト概要

## プロジェクトの目的
電柱番号から位置を検索できる日本初の全国対応無料サービス

## 核心コンセプト
1. **シンプル第一** - 番号札を撮影するだけ
2. **物理と論理の分離** - 1本の柱に複数の番号
3. **ユーザーを信頼** - 自動判定しない
4. **ゲーム感覚** - ランキング、バッジ、MVP賞金

## 市場優位性
- 日本初の全国対応・無料・一般向けサービス
- 競合不在
- 複数の収益源（広告、Pro版、B2B）

## 技術スタック
- Frontend: React 18 + TypeScript + Vite + Capacitor
- Backend: Node.js + Express + Prisma
- Database: PostgreSQL + PostGIS
- 地図: Leaflet.js + OpenStreetMap
- 本番環境: BMAX N95 ミニPC（Ubuntu Server）

## ターゲットユーザー
- 一般ユーザー：電柱位置を知りたい人
- 自治体・町内会：防犯灯などの管理
- 電力会社・通信事業者：設備管理（B2B）
